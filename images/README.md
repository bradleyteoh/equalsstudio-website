# Project photos go here

Put portfolio project photos in this folder, then reference them from
`index.html` like this:

```html
<div class="thumb"><img src="/images/sunway-office.jpg" alt="Sunway corporate office fit-out"></div>
```

Tips:
- Use lowercase file names with dashes, e.g. `sunway-office.jpg` (no spaces).
- Keep photos reasonably sized — under ~500KB each so the site stays fast.
  If a photo is very large, ask Claude Code to note it and Bradley can compress it.
- Always write a short, descriptive `alt=` text for each image.
